# DeepStream Tracker

A real-time video stream monitoring and tracking dashboard. Built to display live tracking data with metadata panels and stream controls.

## Features

- Real-time stream monitoring
- Live tracking display with video statistics
- Stream metadata panel showing current metrics and temperature history
- Stream control interface to start/pause tracking
- Responsive design with Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- ESLint
